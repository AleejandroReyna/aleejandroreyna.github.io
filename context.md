# Project Context: aleejandroreyna.github.io

## Overview
This is a Next.js project bootstrapped with `create-next-app`. It serves as a personal website/portfolio.

## Tech Stack
- **Framework:** Next.js 15.2.4
- **UI Library:** React 19.0.0
- **Styling:** Tailwind CSS 4, DaisyUI 5.0.9
- **Language:** TypeScript
- **Icons:** 
    - FontAwesome (@fortawesome/react-fontawesome)
    - React Simple Icons (@icons-pack/react-simple-icons)
    - Devicons React (devicons-react)

## Available Scripts
- `npm run dev`: Runs the development server (with Turbopack)
- `npm run build`: Builds the application for production
- `npm run start`: Starts the production server
- `npm run lint`: Runs the linter

## Project Structure
- `src/`: Source code directory
- `public/`: Static assets
- `next.config.ts`: Next.js configuration
- `tailwind.config.js`: Tailwind CSS configuration
- `tsconfig.json`: TypeScript configuration
- `package.json`: Project dependencies and scripts
