# Website Structure Plan

## Overview
This document outlines the recommended structure for Alejandro Reyna's personal website (alejandroreyna.com), designed to showcase the unique blend of technical excellence and artistic sensibility that defines the "Code Poet" brand.

## Design Philosophy

### Core Principles
1. **Code as Poetry** - Visual design should reflect artistic sensibility
2. **Technical Excellence** - Showcase full-stack capabilities
3. **Authentic Voice** - Balance professional and creative personality
4. **Modern & Clean** - Reflect current web development trends
5. **Performance First** - Fast, responsive, accessible

### Brand Identity
- **Primary Theme:** Poetic Technologist
- **Color Palette:** Dark mode with vibrant accents (reflecting creativity)
- **Typography:** Modern, clean fonts with artistic touches
- **Tone:** Professional yet creative, technical yet accessible

## Technical Implementation Recommendations

### Technology Stack
**Current Project Stack (Already in use):**
- **Framework:** Next.js 15.2.4 ✅
- **UI Library:** React 19.0.0 ✅
- **Styling:** Tailwind CSS 4 + DaisyUI 5.0.9 ✅
- **Language:** TypeScript ✅
- **Icons:** FontAwesome, React Simple Icons, Devicons ✅

**Additional Recommendations:**
- **CMS:** MDX for blog posts (markdown + React components)
- **Animations:** Framer Motion for smooth transitions
- **Forms:** React Hook Form for contact form
- **Analytics:** Vercel Analytics or Google Analytics
- **SEO:** Next.js built-in SEO optimization

### Performance Goals
- Lighthouse score: 95+ across all metrics
- First Contentful Paint: < 1.5s
- Time to Interactive: < 3.5s
- Mobile-first responsive design

### Accessibility
- WCAG 2.1 AA compliance
- Semantic HTML
- Keyboard navigation
- Screen reader friendly
- Alt text for all images

### SEO Strategy
- Unique meta descriptions per page
- Open Graph tags for social sharing
- Structured data (JSON-LD)
- Sitemap and robots.txt
- Fast page load times

## Visual Design Recommendations

### Color Scheme
**Option 1: Dark Poetic**
- Background: Deep navy/black (#0a0e27)
- Primary: Vibrant purple/blue (#6366f1)
- Accent: Warm gold/amber (#fbbf24)
- Text: Soft white (#f8fafc)

**Option 2: Light Artistic**
- Background: Off-white (#fafafa)
- Primary: Deep indigo (#4f46e5)
- Accent: Creative teal (#14b8a6)
- Text: Rich black (#0f172a)

### Typography
- **Headings:** Modern sans-serif (Inter, Outfit, or Poppins)
- **Body:** Readable sans-serif (Inter or System UI)
- **Code:** Monospace (Fira Code, JetBrains Mono)
- **Accent:** Optional serif for poetic quotes (Playfair Display)

### Visual Elements
- Code snippet animations
- Gradient backgrounds
- Subtle particle effects
- Smooth scroll animations
- Hover effects on interactive elements
- Creative use of whitespace
